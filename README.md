# 🌱 GreenGrow – Smart Fertilizer Recommendation System

A Machine Learning–based web application that recommends the most suitable fertilizer for crops based on soil nutrients and environmental conditions. The system helps farmers and agricultural planners make data-driven decisions to improve crop productivity and soil health.

---

# 📌 Project Overview

GreenGrow is designed to analyze soil nutrient levels such as **Nitrogen (N), Phosphorus (P), and Potassium (K)** along with environmental conditions like **temperature, humidity, and rainfall** to predict the optimal fertilizer recommendation. Systems like this are used in modern precision agriculture to increase crop yield and reduce excessive fertilizer use.

The application provides a **user-friendly web interface** where users can enter soil parameters and receive fertilizer recommendations generated by a trained machine learning model.

---

# 🚀 Features

* 🌾 Fertilizer recommendation using Machine Learning

# Machine Learning Model

The system uses a **Random Forest Classifier** trained on agricultural datasets to predict suitable fertilizers.

### Input Parameters

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* Rainfall
* Soil type
* Crop type

### Model Pipeline

1. Data preprocessing
2. Feature encoding
3. Model training
4. Prediction generation
5. Web application integration

The trained model and encoders are stored as **`.pkl` files** and loaded by the Flask backend during runtime.
