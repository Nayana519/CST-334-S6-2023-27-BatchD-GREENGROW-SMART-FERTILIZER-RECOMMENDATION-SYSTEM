# GREENGROW SMART FERTILIZER RECOMMENDATION SYSTEM

## Project Overview
This project is a smart web application that recommends the appropriate fertilizer based on soil nutrients and environmental conditions.

## Features
- Fertilizer prediction using Machine Learning
- Weather API integration
- Fertilizer quantity calculation
- User profile and prediction history
- Responsive web interface

## Technologies Used
- Python
- Flask
- HTML, CSS, JavaScript
- Machine Learning (Random Forest)
- SQLite Database

## Dataset
The model was trained using datasets from Kaggle and Soil Health Card data.

## How to Run the Project

### 1. Install dependencies
pip install -r requirements.txt

### 2. Train model
python train_model.py

### 3. Run the application
python app.py
